 Start
2. Set `number_to_guess` to RANDOM(1, 100)
3. Set `guess` to 0
4. While `guess` is not equal to `number_to_guess`:
    a. Prompt player to enter a guess
    b. Read `guess`
    c. If `guess` is not a number or `guess` is less than 1 or greater than 100:
        i. Print "Invalid input! Please enter a number between 1 and 100."
    d. Else if `guess` is less than `number_to_guess`:
        i. Print "Too low!"
    e. Else if `guess` is greater than `number_to_guess`:
        i. Print "Too high!"
    f. Else:
        i. Print "Congratulations! You've guessed the correct number."
5. End
